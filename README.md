## 🚀 Overview

A compact and efficient **4-channel relay controller** powered by **ESP32-C3** with **DS3231 RTC**, built for **precise time-based automation of AC loads**.

Designed for real-world applications like **home automation, industrial timers, and smart control systems**.

---

## ✨ Features

* ⚡ **4 × High Power Relays (10A, 250VAC)**
* 🧠 **ESP32-C3 (Wi-Fi + BLE enabled)**
* ⏱️ **DS3231 RTC (high precision timing)**
* 🔒 **Opto-isolated AC sensing**
* 🔁 **Independent channel control**
* 🔌 **Custom compact PCB (KiCad)**
* 🔋 **RTC battery backup support**

## 🧰 Hardware

| Component  | Description                     |
| ---------- | ------------------------------- |
| MCU        | ESP32-C3-MINI-1                 |
| RTC        | DS3231                          |
| Relays     | 10A 250VAC (SPDT)               |
| Input      | 5V DC                           |
| Regulation | LDO 3.3V                        |
| Protection | Flyback diodes + Opto isolation |

---

### Setup

* Power using **5V DC**
* Connect loads to relay terminals
* Flash firmware via **USB-C / UART**
* Configure timers (firmware / web UI)

---

## 🧠 Working Principle

* ESP32 reads **real-time data** from DS3231
* Executes scheduled tasks
* Controls relays independently
* Enables automation of AC devices
